Welcome to the VC Viewer documentation. Here you'll find comprehensive guides and documentation to help you start working with VC Viewer as quickly as possible.

Installation

VC Viewer can be used directly through our hosted version or installed locally. Choose the method that best suits your needs:

Using the Hosted Version

Simply visit vc-viewer.vamaralds.be to start using VC Viewer immediately. No installation required!

Local Installation

To run VC Viewer locally, follow these steps:

# Clone the repository
git clone https://github.com/AmaVic/vc-viewer.git
cd vc-viewer

# Build and run using Docker
docker compose up --build

Quick Start

Get started with VC Viewer in just a few steps:

  1. Navigate to the Viewer page
  2. Paste your Verifiable Credential in the input field
  3. Select a theme from the dropdown menu
  4. Export your visualization as an image or PDF
Note: You can also try the example credentials to see how different themes look.
