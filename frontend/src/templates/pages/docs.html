Getting Started

VC Viewer is a web-based tool for visualizing W3C Verifiable Credentials. Follow these steps to get started:

  1. Navigate to the Viewer page
  2. Paste your W3C Verifiable Credential in JSON format
  3. Choose a theme that matches your credential type
  4. Export the visualization as an image or PDF

Credential Format

VC Viewer supports standard W3C Verifiable Credentials in JSON format. Here's an example:

{
  "@context": [
    "https://www.w3.org/2018/credentials/v1"
  ],
  "type": ["VerifiableCredential"],
  "issuer": "did:example:123",
  "issuanceDate": "2023-01-01T00:00:00Z",
  "credentialSubject": {
    "id": "did:example:456",
    "name": "John Doe"
  }
}

Using Themes

Themes are automatically selected based on your credential type. Available themes include:

  • University Degree (Classic, Modern, LinkedIn)
  • Driver License (Classic, Modern)
  • Default theme for other credential types

Creating Custom Themes

You can create custom themes by extending the BaseTheme class:

class CustomTheme extends BaseTheme {
    constructor() {
        super();
        this.id = "custom-theme";
        this.name = "Custom Theme";
        this.description = "My custom theme";
        this.author = "Your Name";
        this.supportedTypes = ["CustomCredential"];
    }

    render(credential) {
        // Your custom rendering logic
    }
}

API Reference

VC Viewer provides two main API endpoints:

  • /process-credential - Process a single credential
  • /process-presentation - Process a verifiable presentation
