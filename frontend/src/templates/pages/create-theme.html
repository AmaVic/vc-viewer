Creating Themes

Learn how to create custom themes for VC Viewer with a step-by-step guide.

Overview Theme Structure Base Theme Modern University Example Theme Styling Theme Registration Testing Your Theme

Overview

Creating a theme for VC Viewer involves three main components:

  1. A JavaScript class that extends BaseTheme
  2. A CSS file for styling
  3. Theme registration in the application

Theme Structure

Themes are organized in the following directory structure:

frontend/src/themes/
├── base/
│   ├── BaseTheme.js
│   └── GenericTheme.js
└── [CredentialType]/
    ├── classic/
    │   ├── ClassicTheme.js
    │   └── classic.css
    └── modern/
        ├── ModernTheme.js
        └── modern.css

For example, the Modern University theme is located at:

frontend/src/themes/UniversityDegreeCredential/modern/
├── ModernUniversityTheme.js
└── modern.css

Base Theme

All themes extend the BaseTheme class, which provides common functionality:

class BaseTheme {
    constructor(credential) {
        this.credential = credential;
    }

    // Required methods to implement
    getThemeIcon() { }
    getThemeTitle() { }
    render() { }

    // Helper methods available
    formatDate(isoDate) { }
    getIssuerName() { }
    getBaseFooterHTML() { }
}

Modern University Example

Let's look at how the Modern University theme is implemented:

1. Theme Class

Create frontend/src/themes/UniversityDegreeCredential/modern/ModernUniversityTheme.js:

import BaseTheme from '../../base/BaseTheme.js';

class ModernUniversityTheme extends BaseTheme {
    static info = {
        id: 'modern',
        name: 'Modern University',
        description: 'Contemporary design with clean lines',
        author: 'VC Viewer'
    };
    static supportedTypes = ['UniversityDegreeCredential'];

    getThemeIcon() {
        return '';
    }

    getThemeTitle() {
        return this.credential.credentialSubject.degree.type;
    }

    render() {
        const wrapper = document.createElement('div');
        wrapper.className = 'modern-university-theme';
        wrapper.innerHTML = `
            

                ${this.getThemeIcon()}
                
${this.getThemeTitle()}

                
${this.getIssuerName()}

            

            

                

                    

                        
${this.credential.credentialSubject.degree.name}

                        
${this.credential.credentialSubject.degree.type}

                    

                    
${this.renderInfoItems()}

                

            

            ${this.getBaseFooterHTML()}`;
        return wrapper;
    }

    renderInfoItems() {
        const { degree } = this.credential.credentialSubject;
        const items = [
            { label: 'Major', value: degree.major },
            { label: 'Minor', value: degree.minor },
            { label: 'GPA', value: degree.gpa },
            { label: 'Graduation Date', value: this.formatDate(degree.graduationDate) }
        ];
        return items.map(item => `
            

                
${item.label}

                
${item.value || '-'}

            
`).join('');
    }
}

export default ModernUniversityTheme;

Theme Registration

To register your theme, you need to:

1. Add Theme Files

  1. Create a directory for your credential type if it doesn't exist
  2. Add your theme files in a subdirectory (e.g., 'modern' or 'classic')
  3. Include both the JavaScript and CSS files

2. Update HTML Template

Add your theme's CSS to frontend/src/templates/pages/viewer.html:

<!-- Theme CSS -->
<link rel="stylesheet" href="/frontend/src/themes/YourCredentialType/your-theme/style.css">

3. Add Theme Script

Add your theme's JavaScript to frontend/src/templates/pages/viewer.html:

<!-- Theme Scripts -->
<script src="/frontend/src/themes/YourCredentialType/your-theme/YourTheme.js"></script>

4. Add Theme to Gallery

To display your theme in the themes gallery, add it to frontend/src/templates/pages/themes.html:

<div class="theme-card">
    <div class="theme-preview">
        <img src="/frontend/src/images/themes/your-theme-preview.png" alt="Your Theme Preview">
    </div>
    <div class="theme-info">
        <h3>Your Theme Name</h3>
        <p>Your theme description</p>
        <div class="theme-metadata">
            <span class="theme-type">Credential Type</span>
            <span class="theme-author">Author Name</span>
        </div>
    </div>
</div>
Note: Make sure to add a preview image of your theme in the images directory and update the image path accordingly.

Testing Your Theme

To test your theme:

  1. Ensure all files are in the correct locations
  2. Restart the development server
  3. Visit the viewer page
  4. Use an example credential of your theme's type
  5. Select your theme from the dropdown
Tip: Use browser developer tools to inspect and debug your theme's HTML structure and CSS styles.
